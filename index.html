Bridge criticality due to extreme traffic loads

The bridge criticality refers to the ratio between the extreme traffic load effect selected and the characteristic load effect induced by the design live load model. The lowest ratio the better.
Information regarding bridge geometry and site-speciffic traffic characteristics will be popout when select a specific bridge on the map

Nomenclature:

  • M50r load effect performance indicator (ratio) for the bending moment considering a 50-year return period.
  • V50r load effect performance indicator (ratio) for the shear force considering a 50-year return period.
  • M50 Absolute bending moment with a 50-year return period in kNm.
  • V50 Absolute shear force with a 50-year return period in kN.

General Information:

  • Bridge characteristics: Includes details about the name, year of construction, number of lanes, number of spans, maximum span length, and total length of the selected bridge.
  • Site-specific traffic characteristics: Provides information about the Annual Average Daily Truck Traffic (AADTT), the proportion of the traffic flow per vehicle type,
    the design live load model (DLL), and the corresponding value of the performance indicator or extreme load effect.

This information will be displayed in a pop-up window when you select a particular bridge on the map.
